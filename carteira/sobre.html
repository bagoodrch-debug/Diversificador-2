Sobre o Distribui Rico

O Distribui Rico é um simulador gratuito de diversificação de investimentos, feito para quem já tem um valor separado para investir mas não sabe por onde começar a organizar essa distribuição entre categorias.

Para quem é

Para investidores iniciantes e intermediários no Brasil que querem visualizar, de forma simples e visual, como um patrimônio pode ser dividido entre Ações, Fundos Imobiliários, BDRs, Renda Fixa e Ouro — antes de decidir onde efetivamente aplicar o dinheiro.

O que o Distribui Rico não é

  • Não é uma recomendação de investimento. As sugestões são educacionais.
  • Não é uma corretora ou instituição financeira. Nenhuma ordem de compra é executada aqui.
  • Não substitui um profissional certificado. Decisões de investimento reais merecem orientação personalizada.

Como funciona por trás

Toda a simulação roda no seu próprio navegador — os valores que você digita nunca são enviados a um servidor. A única chamada de rede que a ferramenta faz é para buscar cotações públicas de ativos na B3, quando você pede para "buscar cotação" de um ativo específico. Para saber como os percentuais sugeridos são calculados, veja a página de metodologia.

Aviso legal

Este simulador possui finalidade exclusivamente educacional e não constitui recomendação de investimento.
